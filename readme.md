# NodeJS + TypeScript starter project

- Hot-reload
- Linting with ESLint
- Testing with Jest
- Debugging enabled

## Scripts

- `yarn build` to lint and build the app
- `yarn dev` to start the app with hot-reload
- `yarn start` to start the app without hot-reload
- `yarn test` to run all test files
